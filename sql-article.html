Introduction to SQL

What is SQL?

SQL, or Structured Query Language,
is a powerful tool for managing and querying data
in a relational database management system.

What can SQL do?

SQL can perform various operations,
including executing queries,
retrieving data,
inserting records,
updating records,
deleting records, and more.

SQL Syntax :

                
                    SELECT *
                        FROM <Table Name>
                        WHERE <Condition>

The main syntax of SQL includes a SELECT statement,
which is used to query data from a table.
The wildcard (*) is used to select all columns.
The FROM clause specifies the table from which to retrieve data,
and the WHERE clause filters the results based on a specified condition.

Table Example :

NAME AGE CITY
Bob 20 New York
Jane 30 London
Donald 34 Paris

Query Example:

To retrieve data from this table and find students living in London,
we would write the following SQL query : 

                
                    SELECT *
                    FROM Students
                    WHERE City = 'London'

Output Example:

NAME AGE CITY
Jane 30 London
